# Metasploit challenge project

# Overview
This was a challenge project where I set out to learn the basics of developing exploit modules for the Metasploit Framework. Exploit development is a vast field, and I haven't even begun to explore low-level binary exploitation or exploiting buffer overflows. However, this was an important first step towards learning Ruby, specifically for exploit development for the Metasploit Framework. 

## Update
I decided to revisit this project, and have improved the exploit by spawning a native Metasploit shell. This allows the shell to appear as a session when using the `sessions` command, and allows you to background the shell for later use. This has helped me gain a better grasp on how the Metasploit library works and is a valuable learning experience.
